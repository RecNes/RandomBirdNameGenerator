About

The Website Service


This is the free service for who wants to stay anonymous, but lazy to create a nick name. Random Bird Name Generator is exactly for those people who are do not want waste their time to create an anonymous nickname. Just press button above, and Random Bird Name Generator does the magic for you and give you a nickname. Then start to use your anonymous name where ever you want.

The API Service (Coming Soon!)


The API of this service is still under development. The websites which want to be provide anonymity to their new registered users can be able to use this API. During the registration process, you can hit to the API before creating user account and get a random bird name as a nickname for new registrations.
